About CS Clicker

The Story So Far...

Coding styles have been a common point of contention amongst computer scientists and software engineers alike. After starting a new position as a junior software engineer, you find that the topic is a common occurance in your workplace, usually introduced in a somewhat confrontational manner. In this endless syntactical struggle of where (or whether) to place semicolons, naming variables appropriately, and other oddities, you finally had enough and decided to do something about it.

The main objective of this game is to generate lines of code which then can be used to help feed data to a newly-developed machine learning algorithm of your own design. This algorithm could then be used to generate the perfect coding style so everyone can finally agree on one coding standard. As more and more lines of code are generated, it gains the attention of local user groups in the area, who start contributing their own code. This eventually accelerates to the point where people are willing to donate machines hardware such as servers and even quantum computers to generate code for the cause of creating the perfect coding style.
